# ABIDS - Automated Business Intelligence Dashboard

A Streamlit-powered dashboard that analyzes business data, detects profit leakage, and generates automated recommendations.

## Features
- KPI computation
- Profit leakage detection
- Recommendation engine
- Executive dashboard

## Tech Stack
- Python
- Pandas
- Streamlit

## How to Run
pip install -r requirements.txt
streamlit run Dashboard/app.py
